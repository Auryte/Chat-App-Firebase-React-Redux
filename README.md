# Chat App Firebase

This is a realtime one to one chat app made using Firebase, React, Redux.

# Features

* Firebase:
  - Authentication (Email)
  - Cloud Firestore
  - Cloud Messaging
  - Functions
  - Remote Config
  - Storage
* Create user profile (Username).
* Send text.
* Take a picture from gallery.
* Check user online status.
